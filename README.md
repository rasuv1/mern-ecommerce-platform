# MERN E-commerce Platform

A single-vendor e-commerce web application built using the MERN stack.  
This project focuses on secure authentication, product management, cart and order flow, and an admin dashboard.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication

## Features (Planned)
- User authentication
- Product listing & details
- Cart & checkout
- Order management
- Admin dashboard
